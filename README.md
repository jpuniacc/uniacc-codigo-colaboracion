# 📖 Código de Colaboración UNIACC

Decálogo digital con los 7 principios de colaboración de UNIACC.

## 🚀 Instalación

```bash
git clone https://github.com/jpuniacc/uniacc-codigo-colaboracion.git
cd uniacc-codigo-colaboracion
npm install
npm run dev
```

## 📱 Uso

- **Home** (`/`) - Ver los 7 principios
- **Principio** (`/principios/:id`) - Ver detalles de cada principio

## 🎯 Los 7 Principios

1. **👁️ UNIACC Consciente** - Presencia plena en reuniones
2. **⏰ UNIACC Puntual** - Gestión eficaz del tiempo
3. **💬 UNIACC Comunica** - Comunicación clara
4. **🤝 UNIACC Valora** - Diversidad e inclusión
5. **🔧 UNIACC Resuelve** - Gestión de conflictos
6. **💡 UNIACC Co-crea** - Innovación colaborativa
7. **🌟 UNIACC Reconoce** - Cultura de apreciación

## 🛠️ Tecnologías

- Vue.js 3 + Vite
- PWA
- CSS Variables
- Sin dependencias complejas

## 🚀 Deploy

```bash
npm run build
# Subir carpeta 'dist' a Netlify
```
